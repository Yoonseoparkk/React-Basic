# React-Basic : Unsplash Image API를 활용한 이미지 검색 사이트 만들기

## 개발 환경
1. 프로젝트 환경설정(vite를 활용한 React 설치): <code>npm install vite@latest</code>
2. React 중앙집중식 상태관리 라이브러리 Recoil 설치: <code>npm install recoil</code>
3. 외부 오픈 API 통신을 위한 라이브러리 Axios 설치: <code>npm install axios</code>
4. CSS 스타일링을 위한 SASS/SCSS 설치: <code>npm install -D sass</code>
5. React Router 설치: <code>npm install react-router-dom localforage match-sorter sort-by</code>
6. TypeScript에서 Node.js 모듈을 쓸 수 있는 환경 구축 : <code>npm i @types/node</code>
7. React Toast Popup 모듈 설치 : <code>npm install react-simple-toasts</code>
